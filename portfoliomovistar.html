Proyecto: App Movistar

La App Movistar es una solución digital que facilita la gestión de servicios de telecomunicaciones. Los usuarios pueden consultar su saldo, realizar pagos, monitorear consumos y contratar planes adicionales, todo desde una interfaz intuitiva y amigable. Participé en el diseño y desarrollo de la app, asegurando una experiencia de usuario fluida y accesible.

Con este proyecto, se buscó optimizar el servicio al cliente, permitiendo una mayor autonomía y conveniencia para los usuarios de Movistar.

Proyecto: Mi Movistar - Telefónica

Rol: UI Developer y Front-end Developer

Tecnologías utilizadas: HTML, SCSS, Angular, Ionic.

En el proyecto Mi Movistar, desempeñé un papel clave en el desarrollo de aplicaciones y sitios web para Telefónica. Mis responsabilidades incluyeron:

  • Prototipado Ágil: Desarrollo rápido de la aplicación Mi Movistar usando HTML, SCSS y Xcode para asegurar un diseño responsivo en Android e iOS.
  • Desarrollo Front-end: Creación de la Sucursal Virtual Mi Movistar con HTML, SCSS, Angular 2, Ionic y JavaScript, garantizando una experiencia web fluida.
  • Design System: Participación en la creación de un Design System con componentes reutilizables, alineado a los estándares de Telefónica España.
  • Colaboración Ágil: Trabajo en equipo bajo metodología Scrum, utilizando Jira y Git para gestión de proyectos.

Habilidades Desarrolladas:

  • Dominio de tecnologías web y móviles.
  • Fomento de colaboración y gestión de proyectos ágiles.

Este proyecto fortaleció mis habilidades técnicas y mi capacidad para mejorar la experiencia del usuario en telecomunicaciones.
